# VS Code

## 자주 사용하는 단축키
* 반복되는 코드를 한번에 정리하기
    * mac: `Command + D`
    * Windows: `CTRL + D`
* 코드 위/아래로 이동하기
    * mac: `Option + ↑ 또는 ↓`
    * Windows: `Alt + ↑ 또는 ↓`
* 코드 위/아래로 복사하기
    * mac: `Shift + Option + ↑ 또는 ↓`
    * Windows: `Shift + ↑ 또는 ↓`
* 코드를 한번에 주석 처리하기
    * mac: `Command + /`
    * Windows: `Ctrl+ /`
* 여러 곳에 커서를 두고 수정하기
    * mac: `Option + 마우스 클릭`
    * Windows: `Alt + 마우스 클릭`
* 다중 커서 선택하여 수정하기
    * mac: `Command + Option + ↑ 또는 ↓`
    * Windows: `Ctrl + Alt + ↑ 또는 ↓`
* 일치하는 중괄호로 이동( :thumbsup: )
    * mac: `Command + Shift + \`

## 터미널 현재 위치에서 vscode 열기
### mac
1. Launch VS Code
2. Open the Command Palette (Ctrl+Shift+P) and type 'shell command' to find the Shell Command: Install 'code' command in PATH command.
3. 터미널 열고 원하는 위치에서 `code .` 입력
### Windows
터미널 열고 `code .`입력  
(Tip. 탐색기 주소창에 cmd 입력하면 해당 위치에서 cmd창 열림)
